# Aplicacion Python Flask

## Requisitos
* Python
* Pip

## Como Ejecutar la Aplicacion

1. Cree un ambiente virtual

### Linux
```bash
python3 -m venv venv
```

### Windows
```
python -m venv venv
```

2. Active el ambiente virtual

### Linux
```bash
source venv/bin/activate
```

### Windows
```
.\venv\Scripts\activate
```

3. Instalar las dependencias Python
```bash
pip install -r requirements.txt
```

4. Ejecutar la Aplicacion
### Linux
```bash
python3 ./main.py
```

### Windows
```
python.exe .\main.py
```
